  1. 表单标签
1. input标签：

可以根据type属性值的不同，展示不同效果。

type属性值：

标签名 type属性值 说明
input text 文本框，用于输入单行文本
input password 密码框，用于输入密码
input radio 单选框，用于多选一
input checkbox 多选框，用于多选多
input file 文件选择，用于之后上传文件
input submit 提交按钮，用于提交
input reset 重置按钮，用于重置
input button 普通按钮，默认无功能，之后配合js添加功能

placeholder属性值：给文本框加上提示字

radio：单选功能和默认选中

file：默认单文件选择，多文件选择需要加上属性值multipe

按钮标签：submit、reset、button

表单域标签：使按钮功能正常需要为实现区域添加form标签

value属性：为按钮添加显示文字

2. button标签：

可以根据type属性值的不同，展示不同效果。

type属性值：

标签名 type属性值 说明
button submit 提交按钮，用于提交数据给后端服务器
button reset 重置按钮，用于重置表单默认值
button button 普通按钮，默认无功能，之后配合js添加功能

button是双标签，更便于包裹其他内容：文字、图片等

3. select标签：

select标签：下拉菜单的整体

option标签：下拉菜单的每一项

常见属性：selected：下拉菜单的默认选中
